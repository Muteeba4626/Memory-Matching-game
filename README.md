# FLIPCARD — Memory Matching Game

A simple memory card game built with HTML, CSS, and JavaScript.

## How to Play

1. Click a card to flip it and see the emoji
2. Click another card to find its match
3. If both emojis match, they stay flipped ✅
4. If not, they flip back ❌
5. Match all 8 pairs to win 🏆

## How to Run

1. Download all 3 files into the same folder:
   - `index.html`
   - `style.css`
   - `game.js`
2. Double-click `index.html` to open it in Chrome

> Make sure you have an internet connection the first time — the game uses jQuery from a CDN.

## Features

- Flip animation on cards
- Move counter and timer
- Best score tracking
- Restart button

## Built With

- HTML, CSS, JavaScript
- jQuery
- Google Fonts (Orbitron, DM Mono)
